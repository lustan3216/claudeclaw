---
name: goclaudeclaw:install
description: Install the goclaudeclaw binary
---

Download and install the goclaudeclaw binary for your platform.

Run the install script:
```sh
curl -fsSL https://raw.githubusercontent.com/lustan3216/goclaudeclaw/main/install.sh | sh
```

After installation, copy `config.example.yaml` to `~/.goclaudeclaw/config.yaml` and add your Telegram bot tokens.
